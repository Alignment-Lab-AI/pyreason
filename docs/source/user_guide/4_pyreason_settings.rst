
PyReason Settings
=================


+-----------------+------------------+-------------------------------------------------------+
| Setting         | Default          | Description                                           |
+-----------------+------------------+-------------------------------------------------------+
| verbose         | True             | Returns whether verbose mode is on                    |
+-----------------+------------------+-------------------------------------------------------+
| output to file  | False            | Returns whether output is going to be printed to file |
+-----------------+------------------+-------------------------------------------------------+
